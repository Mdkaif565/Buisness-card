# Business Card

This project is a simple business card webpage created using HTML and CSS.  
It displays essential contact details and includes one image used as part of the design.

## Files Included
- `index.html` — Structure and content of the business card
- `style.css` — Styling and layout design
- `image.jpg` — Image used in the business card (e.g., logo or photo)

## How to View
1. Make sure all three files are in the same folder.
2. Open **index.html** in any web browser.

## Purpose
This project demonstrates a basic static web design for a business card layout.
